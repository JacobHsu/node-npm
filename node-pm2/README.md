P(rocess) M(anager) 2 Runtime Edition   

PM2 是 Node.js 應用程式的正式作業程序管理程式，具有一個內建的負載平衡器。PM2 容許您讓應用程式永遠維持作用中，並在重新載入時不需關閉，且能協助進行一般的系統管理作業。PM2 也可讓您管理應用程式記載、監視和叢集作業。  

如需相關資訊，請參閱 https://github.com/Unitech/pm2。
